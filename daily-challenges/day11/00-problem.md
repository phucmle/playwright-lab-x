# JavaScript

## Problem:

Write a function to calculate the tip amount based on the total bill and tip percentage.

### Requirements:

- Write a JavaScript function called `calculateTip` with 2 parameters: total bill and tip percentage(%) to calculate the
  tip amount.

## Example:

**Input**:

- Total bill: 100
- Tip percentage: 15%

**Output**:

- Tip amount: 15
